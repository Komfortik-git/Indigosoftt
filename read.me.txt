Node.js (версия >= 14.x)
npm (>= 6.x)
Git 

Клонируйте репозиторий:
git clone https://github.com/Komfortik-git/Indigosoftt.git
cd your-repository

Установите зависимости: npm install

Запустите тесты в интерактивном режиме: npx cypress open

Появится интерфейс Cypress, где вы сможете выбрать нужный тест и наблюдать его выполнение в браузере.

Если вы хотите запускать тесты без графического интерфейса, 
выполните команду: npx cypress run
